# 快速启动指南（无需Maven命令行）

## 方法一：使用IntelliJ IDEA（推荐）

### 步骤1：打开项目
1. 打开IntelliJ IDEA
2. File -> Open -> 选择项目文件夹 `C:\Users\xx\Desktop\1`
3. 等待IDEA自动识别Maven项目并下载依赖（首次可能需要几分钟）

### 步骤2：配置数据库
1. 确保MySQL服务已启动
2. 打开 `src/main/resources/application.yml`
3. 修改数据库连接信息：
```yaml
spring:
  datasource:
    username: root        # 你的MySQL用户名
    password: root        # 你的MySQL密码
```

### 步骤3：创建数据库
在MySQL中执行：
```sql
CREATE DATABASE IF NOT EXISTS hospital_db DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
USE hospital_db;
```
然后执行 `src/main/resources/db/schema.sql` 文件中的SQL语句。

### 步骤4：运行项目
1. 找到 `src/main/java/com/hospital/HospitalManagementSystemApplication.java`
2. 右键 -> Run 'HospitalManagementSystemApplication'
3. 等待启动完成（控制台显示 "Started HospitalManagementSystemApplication"）

### 步骤5：访问系统
1. 打开浏览器
2. 访问：http://localhost:8080/hospital
3. 使用默认账户登录：
   - 用户名：admin
   - 密码：admin123

同IntelliJ IDEA的步骤2-5

## 方法三：安装Maven后使用命令行

如果你需要安装Maven，请参考 `Maven安装指南.md` 文件。

安装完成后：
```powershell
cd C:\Users\xx\Desktop\1
mvn clean install
mvn spring-boot:run
```

## 常见问题

### 1. IDEA提示"找不到Maven"
- File -> Settings -> Build, Execution, Deployment -> Build Tools -> Maven
- 检查Maven home directory是否正确
- 如果没有Maven，IDEA会使用内置的Maven

### 2. 依赖下载失败
- 检查网络连接
- 尝试更换Maven镜像源（在IDEA的Maven设置中）

### 3. 数据库连接失败
- 确认MySQL服务已启动
- 检查用户名密码是否正确
- 确认数据库hospital_db已创建

### 4. 端口被占用
- 修改 `application.yml` 中的 `server.port: 8081`（改为其他端口）
