# 立即启动项目 - 最简单方法

## ✅ 当前状态检查

- ✅ Java 17 已安装
- ✅ MySQL 服务正在运行
- ✅ 项目已编译（target目录存在）

## 🚀 启动方法（推荐）

### 方法一：在IntelliJ IDEA中启动（最简单）

1. **打开项目**（如果还没打开）
   - File → Open → 选择 `C:\Users\xx\Desktop\1`

2. **找到主类**
   - 在项目树中找到：`src/main/java/com/hospital/HospitalManagementSystemApplication.java`

3. **运行项目**
   - **右键点击** `HospitalManagementSystemApplication.java`
   - 选择 **"Run 'HospitalManagementSystemApplication'"**
   - 或点击类名旁边的 **绿色运行按钮**（▶️）

4. **查看启动日志**
   - 在IDE底部的控制台（Console）查看
   - 等待看到：`Started HospitalManagementSystemApplication`

5. **访问系统**
   - 打开浏览器
   - 访问：**http://localhost:8080/hospital**
   - 登录：admin / admin123

## ⚠️ 如果启动失败

### 检查1：查看控制台错误信息

启动时如果看到红色错误，常见原因：

**错误A：数据库连接失败**
```
Communications link failure
```
**解决**：
1. 确认MySQL服务运行中（已确认 ✓）
2. 检查数据库是否创建：
   ```sql
   CREATE DATABASE hospital_db;
   ```
3. 检查 `application.yml` 中的用户名密码

**错误B：端口被占用**
```
Port 8080 is already in use
```
**解决**：修改 `application.yml` 中的端口为8081

**错误C：编译错误**
**解决**：Build → Clean Project → Rebuild Project
