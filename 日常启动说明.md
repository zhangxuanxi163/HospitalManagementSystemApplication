# 日常启动说明

## ✅ 是的，下次直接启动Java源代码即可！

## 🚀 启动步骤（每次启动都这样做）

### 1. 确保MySQL服务运行
- 通常MySQL服务会自动启动
- 如果没运行，手动启动MySQL服务

### 2. 在IDE中运行项目
- 打开IntelliJ IDEA
- 找到：`src/main/java/com/hospital/HospitalManagementSystemApplication.java`
- 右键 → **Run 'HospitalManagementSystemApplication'**
- 或点击绿色运行按钮（▶️）

### 3. 等待启动完成
- 查看控制台，看到：
  ```
  Started HospitalManagementSystemApplication in X.XXX seconds
  ```

### 4. 访问系统
- 打开浏览器
- 访问：**http://localhost:8080/hospital**
- 登录：**admin / admin123**

---

## ⚠️ 注意事项

### 第一次启动后，数据库已创建
- ✅ 数据库 `hospital_db` 已创建
- ✅ 表结构会自动创建（Hibernate会自动处理）
- ✅ 默认管理员账户已创建

### 如果启动失败，检查：

1. **MySQL服务是否运行**
   - 打开"服务"（services.msc）
   - 确认MySQL80服务状态为"正在运行"

2. **数据库连接配置**
   - 检查 `src/main/resources/application.yml`
   - 确认用户名密码正确

3. **端口是否被占用**
   - 如果8080被占用，修改 `application.yml` 中的端口
